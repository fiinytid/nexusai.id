Cara Kerja

3 Langkah Simple

Setup sekali, pakai selamanya.

1
Login & Setup
Login dengan Google + Roblox. Install plugin NEXUS AI di Roblox Studio. Download dari Discord kami.
2
Connect Plugin
Buka Roblox Studio, klik NEXUS AI di toolbar, klik CONNECT. Status berubah hijau = siap!
3
Chat & Build
Ketik apa yang mau dibuat. AI langsung inject code + object ke place kamu. Sesimple itu!
// AI Response → JSON command langsung ke Studio
// AI akan membuat house dengan batch_commands
{
"action": "batch_commands",
"commands": [
{"action":"create_part","name":"Floor","size":[20,1,20]},
{"action":"inject_script","name":"GameSystem","parent":"ServerScriptService"}
]
}
✅ Plugin received 2 commands → Floor created, GameSystem injected!
